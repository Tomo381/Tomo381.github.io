# ポートフォリオサイト README

## 📌 プロジェクト概要
HTML / CSS / JavaScript を用いて制作する、紺色ベースのスタイリッシュなポートフォリオサイト。  
就職活動や研究成果の提示、制作物のリンク共有を目的とする。

---

## ✅ 要件定義

### 目的
- 就職活動・研究成果の提示・制作物のリンク共有
- 採用担当者や研究仲間、クライアントに向けた自己PR

### 対象ユーザー
- 採用担当者
- 研究仲間
- クライアント

### 必須機能
- 自己紹介（プロフィール）
- スキル一覧
- 研究成果・制作物リンク集
- お問い合わせ（メールやSNSリンク）

### デザイン要件
- 紺色ベース（背景やヘッダーに使用）
- スタイリッシュでシンプル（余白を活かす）
- レスポンシブ対応（スマホでも見やすい）
- スクロールアニメーション（要素がふわっと浮かび上がる）

---

## 🎨 デザイン構成

1. **ヘッダー**
   - サイト名 / ナビゲーション（Home, About, Works, Contact）
   - 紺色背景＋白文字

2. **トップセクション**
   - キャッチコピー（例: "UI/UX Designer & Web Developer"）
   - シンプルな自己紹介文

3. **About（自己紹介）**
   - 写真（丸型）＋テキスト
   - スキル一覧（リスト形式）

4. **Works（研究成果・制作物）**
   - カード型レイアウト
   - 各カードに「タイトル」「簡単な説明」「リンクボタン」
   - 🔗 制作物のWEBリンクをカード内の `<a>` タグ部分に貼る

5. **Contact（お問い合わせ）**
   - メールリンク、SNSアイコン

6. **フッター**
   - コピーライト表記

---

## ⚙️ 技術要件

- **HTML**
  - セマンティックなタグ構造
  - セクション分割（header, section, footer）

- **CSS**
  - 紺色ベースのスタイリッシュデザイン
  - レスポンシブ対応（`@media` クエリ）
  - ホバー時のアニメーション効果
  - スクロールアニメーション用クラス（`.fade-in`）

- **JavaScript**
  - Intersection Observer API を用いたスクロールアニメーション
  - 要素が画面に入った際に `.show` クラスを付与
